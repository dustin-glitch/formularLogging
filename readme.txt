Installable logging plugin for WordPress form and mail pipelines.

== Description ==
Tracks form submit events, frontend errors, mail pipeline events and SMTP results into daily CSV logs.

== Installation ==
1. Zip the folder `formular-logging`.
2. Upload under Plugins > Add New > Upload Plugin.
3. Activate the plugin.
4. Configure form pages under Settings > Formular Logging.
